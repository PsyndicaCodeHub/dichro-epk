## Deploy

This site is deployed via GitHub Pages with a custom domain:

1. Go to **Settings** → **Pages**
2. Set Source to **Deploy from a branch**
3. Select **main** branch, **/ (root)** folder
4. Click **Save**
5. Under **Custom domain**, enter `dichromusic.com` and click Save

GitHub will auto-detect the `CNAME` file and serve the site at `dichromusic.com`.
